Detailed Description

The logic of the game.

This class contains the main logic fields and functions Public constants are for the sizes of game field and for counters(game objects) The class has protected fields: the array of counters, emty counter and logical variable for finishing the game Public methods: Init- fill the game field, Check- checks is the game solved, Move- movings in the game, two setters
